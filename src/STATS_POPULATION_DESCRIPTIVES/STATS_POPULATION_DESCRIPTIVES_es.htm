Estadísticas descriptivas de población

Produce N, media, desviación estándar y varianza para variables numéricas, suponiendo que los datos representan una población completa en lugar de una muestra. Las desviaciones estándar y las varianzas utilizan N en los denominadores en lugar de N-1.


Requisitos de datos y operaciones:

Las variables de entrada se deben definir como Numéricas.

No se permiten archivos segmentados. Las divisiones se desactivan.
