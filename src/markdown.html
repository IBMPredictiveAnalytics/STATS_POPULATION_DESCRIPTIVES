STATS POP DESCRIPTIVES

Generate Descriptive Statistics for a Population

STATS POP DESCRIPTIVES
VARIABLES = list of variable names*
MISSING = VARIABLE** or LISTWISE

* Required
** Default

STATS POP DESCRIPTIVES /HELP prints this information and does nothing else.

STATS POP DESCRIPTIVES X Y Z1 TO Z10.

String variables will be automatically excluded.

The output includes N, min, max, mean, standard deviation, and variance.

The values will be the same as for the DESCRIPTIVES procedure except that the standard deviation and variable use N as the denominator instead of N-1.
